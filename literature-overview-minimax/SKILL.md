# Literature Overview Skill

## 概述

此技能用于阅读 PDF 格式的学术文献，生成符合中文阅读习惯的文献概览。专门服务于天体物理、天文技术与方法领域的学术论文。

## 工作流程

### 步骤 1: 获取用户输入

当用户输入 `/literature-overview-minimax` 时，执行以下操作：

1. 礼貌问候用户
2. 询问要读取的 PDF 文献的完整路径（包括文件名）
3. 等待用户提供路径

### 步骤 2: 读取 PDF 文献

使用 PDF 技能读取用户提供的 PDF 文件。提取以下内容：

1. **标题** — 论文完整标题
2. **作者** — 所有作者姓名
3. **发表信息** — 发表时间、期刊/会议名称、卷号、页码、DOI
4. **摘要** — 完整摘要内容
5. **正文** — 全部正文内容
6. **参考文献** — 完整的引用文献列表
7. **图片** — 所有图片（带图注）
8. **表格** — 所有表格（带表题）

### 步骤 3: 信息提取与翻译

#### 3.1 标题和摘要翻译

- 如果文献是**英文**：
  - 将标题完整翻译为中文
  - 将摘要完整翻译为中文
  - 如果对翻译内容把握不准，在翻译后附上英文原文
- 如果文献已是**中文**：
  - 标题保持原样
  - 摘要保持原样

#### 3.2 提取文献信息

从文献中提取以下信息，**必须忠于原文**，不能引申、发挥、杜撰：

1. **基本信息**
   - 作者（中文译名+英文原文）
   - 发表时间（年份）
   - 发表刊物/会议
   - DOI

2. **研究背景**
   - 该研究所属领域的发展现状
   - 存在的问题或未解决的问题

3. **研究意义**
   - 该研究的理论意义
   - 该研究的实际应用价值

4. **研究目标**
   - 研究要解决的具体问题
   - 研究要达成的具体目标

5. **研究方法**（重点）
   - 使用了什么仪器/设备
   - 数据来源和处理方法
   - 分析方法/模型
   - 实验设计

6. **主要结论**（重点）
   - 研究得到的主要发现
   - 关键数据支撑

7. **创新点**（重点）
   - 方法创新
   - 理论创新
   - 应用创新

8. **研究展望**
   - 未来研究方向
   - 待解决的问题

### 步骤 4: ADS 引文格式

生成符合 ADS（Astrophysics Data System）格式的标准引文：

**格式**：
```
作者, 年份. 标题. 期刊, 卷号, 页码. DOI
```

**示例**：
```
Smith, J., & Johnson, M. 2023. A New Method for Galaxy Classification.
Astronomical Journal, 165, 42. https://doi.org/10.3847/1538-3881/aca123
```

使用 docx 技能生成 Word 文档，格式如下：

### 步骤 6: 处理图片、表格和公式

1. **图片处理**：
   - 从 PDF 中提取所有图片
   - 将图片插入到 Word 文档中对应的文字描述附近
   - 保留原图的图注

2. **表格处理**：
   - 从 PDF 中提取所有表格
   - 将表格插入到 Word 文档中对应的文字描述附近
   - 保留原表的表题

3. **公式处理**：
   - 如果可能，将公式作为图片插入
   - 或者用文字描述公式的含义

**重要**：图片、表格、公式必须直接嵌入在 Word 文档中对应文字的附近，不能存放在单独的文件夹中。

## 注意事项

1. **忠实原文**：所有提取的信息必须忠于原文，绝不能杜撰、引申或发挥
2. **重点突出**：研究方法、主要结论、创新点是重点描述部分，需要详细阐述
3. **图片对应**：确保图片、表格与其说明文字在同一页面或紧邻的位置
4. **引用完整**：原文中的引用文献需要完整保留在参考文献中
5. **天体物理领域**：此技能专注于天体物理和天文技术与方法领域的文献

## 输出

生成的 Word 文档保存为 `文献概览_原文件名.docx`，其中"原文件名"是输入PDF文件的名字（不含.pdf扩展名）
